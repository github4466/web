Project. 项目
  • 移动: 仿饿了么
    Demo: 左方二维码Code : www.github.com/github4466/sell

    vue2.0 + vue-resource + vue-router + vue-cli + better-scroll。通过webpack的自动压缩合并、图标字体和小图片的base64转换，以及vue-router减少页面HTTP请求；通过vue-resource与后端交互；利用better-scroll做页面的弹性滚动；利用scss、autoprefixer实现对css的处理；利用vue的组件化和数据驱动特点，能够集中精力在业务逻辑，方便地开发饿了么APP单页应用。

    页面主要有头部信息公告栏、中间的切换显示部分、底部购物车。头部点击公告会显示公告栏，公告栏中关闭按钮利用了sticky footer布局；中间部分可做弹性滚动，商品栏的两侧滚动互相关联，可添加商品到购物车，点击食物图片可查看详细信息。评价栏可以筛选评价类型。商家栏有收藏功能，实景图可横向滚动；底部购物车点击后能显示订购信息。

    整个页面实现了CSS的常见居中布局、flex布局、sticky footer布局、一像素边框等。商家、商品信息通过vue-resource实时从后端获取。整个页面功能完善。

  • PC: 仿阅文集团
    Demo: www.dogday.xyz/yuewen Code : www.github.com/github4466/yuewen

    jquery + gulp + scss。页面多处利用雪碧图，单色小图标制作为图标字体文件，以此减少图片的HTTP请求数。jquery尽量只修改DOM元素类名，提高效率。

    首页实现了常见的轮播图，首页的滚动和头部标签栏是互相关联的。点击新闻中心的查看详情按钮，以Ajax的方式从后端获取数据来进行显示。点击头部的加入我们按钮，即可跳转到招聘页面，可以在招聘页面的搜索框输入岗位名称和选择招聘渠道来搜索招聘信息。

  • PC: 电影院
    Demo: www.dogday.xyz/cinema Code : www.github.com/github4466/cinema

    原生JS + gulp + scss。利用CSS3的3D变换，实现了在不同的电影院座位观看电影的效果预览。点击视频下方的选择座位按钮，即可进入影院模式，在影院模式下点击窗口中心的旋转角度按钮可实现180度旋转，在窗口右上方的的座位区域选择座位，即可预览在此座位下观看视频的效果。

    由于利用了CSS3的3D变换，所以旧浏览器可能无法呈现或出现卡顿，在新版的webkit浏览器下浏览效果较好。
